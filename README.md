#  FinalCegepMobileApp (Global currencies) – Application de conversion de devises

##  Description du projet

Le concept de notre application mobile est de permettre facilement à l’utilisateur de **comparer et convertir des devises**.  
L’objectif principal de l’application est d’offrir une **interface intuitive et efficace** permettant :

- d’entrer un montant,
- de sélectionner les devises souhaitées,
- d’obtenir rapidement une conversion **claire, précise et fiable**, basée sur des **taux de change à jour**.

L’application vise à simplifier la conversion de devises sans la complexité des outils financiers traditionnels.  
Dans un contexte lié aux **voyages**, aux **achats en ligne internationaux** et aux **échanges économiques**, cet outil permet à l’utilisateur d’obtenir rapidement l’information nécessaire pour effectuer des conversions.

De plus, l’utilisateur peut **s’enregistrer**, **consulter l’historique de ses conversions** et **ajouter une photo de profil** afin de personnaliser son expérience.

---

##  Objectifs du projet

- Faciliter la conversion de devises en temps réel  
- Offrir une interface simple et ergonomique  
- Permettre la gestion d’un compte utilisateur  
- Conserver l’historique des conversions  
- Appliquer les notions vues en développement mobile au cégep  

---

##  Fonctionnalités principales

-  Conversion de devises en temps réel  
-  Sélection de devises (CAD, USD, EUR, etc.)  
-  Entrée de montant simple et rapide  
-  Création de compte utilisateur  
-  Historique des conversions  
-  Ajout d’une photo de profil  
-  Interface mobile intuitive  

---

##  Technologies utilisées

- React Native  
- Expo  
- JavaScript  
- API de taux de change  (FxRate)

##  Installation et exécution

### Prérequis

- Node.js  
- Expo CLI  
- Un émulateur Android / iOS ou un téléphone cellulaire 

### Étapes

```bash
git clone https://github.com/MarekGromko/FinalCegepMobileApp.git
cd FinalCegepMobileApp
npm install
npx expo start
```
- scanner  le code QR avec le téléphone cellulaire
